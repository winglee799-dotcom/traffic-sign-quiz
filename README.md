# 交通标志题库

## 目录用途

- `31-50.html`：正式考试版。这个链接可以直接发给她用。
- `v2/31-50.html`：迭代工作台。以后继续加题、改题、试新逻辑，先改这里。
- `index.html`：入口页，方便你自己区分正式版和迭代版。

## 现在怎么用

1. 发给她正式版链接：
   `https://winglee799-dotcom.github.io/traffic-sign-quiz/31-50.html`
2. 你自己要继续改题，就先打开迭代版：
   `https://winglee799-dotcom.github.io/traffic-sign-quiz/v2/31-50.html`
3. 等迭代版稳定后，再把改动同步到正式版。

## 更新方式

你以后只要：

1. 修改 `v2/31-50.html`
2. `git add`
3. `git commit`
4. `git push`

GitHub Pages 会自动重新发布，通常几分钟内生效。
